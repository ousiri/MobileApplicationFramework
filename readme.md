
# Env

```sh
npm install -g jello
npm install -g fis-parser-less fis-parser-coffee-script fis-parser-jade fis-postprocessor-pleeease fis-prepackager-csswrapper fis-prepackager-ousiri-async-build
```

# Run

## on local server

```sh
cd src
jello server start
jello release -wL
```

## build to release

```sh
cd src
jello release -ompDd dist
```
